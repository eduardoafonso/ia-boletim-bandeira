Arquitetura Recomendada

Camada de Conhecimento

Fonte Única: SharePoint

Visa, Mastercard, Elo, Amex, Manuais POS, Layouts Batch e Docs Internos.

Camada de Ingestão

C# + .NET / Azure Functions

Azure AI Document Intelligence para extração de tabelas, anexos, datas e regras.

Camada Vetorial

Azure AI Search

Indexação por Bandeira, Mandatoriedade e Áreas Impactadas (Settlement, Chargeback).

Camada Foundry

Diferencial: Ecossistema de agentes especializados em vez de um chatbot genérico.

Agente 1: Relevância

Filtra Adquirência (POS, Batch, Conciliação).

Agente 2: Mandatoriedade

Detecta prazos e flags de Compliance.

Agente 3: Impacto Técnico

Classifica domínios de impacto (Alto/Baixo).

Agente 4: Caso de Uso

Gera Fluxo Futuro e Critérios de Aceite.

Agente 5: Especialista PL/1

Sugere alterações em Copybooks e JCLs.

Governança & Auditoria

  • Identidade: Microsoft Entra ID (SSO/RBAC).
  • Auditoria: Azure SQL (Quem, quando e o que consultou).
  • Observabilidade: Azure Monitor e Prompt Versioning no Foundry.

Interface (UI/UX)

Recomendação: Arquitetura Híbrida (React para Dashboards + Copilot Studio para Teams).

Next.js | TypeScript | Tailwind | C# API
